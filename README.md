This is an interactive light show I built with p5.js. Toggle the switch to watch colorful lights flash randomly while the background blends their colors. This was originally created for my pre-work assignment in my application to Code the Dream.

## Live Demo
Check it out here: [Party Lights](https://joulesjetson.github.io/party-lights/)

## Features
- Toggle switch to turn lights on/off
- 4 randomly flashing lights (purple, yellow, pink, blue)
- Dynamic background that blends the colors of active lights

## How to Use
1. Click on the switch to turn it ON
2. Watch the lights flash randomly
3. Notice how the background color changes based on which lights are on
4. Click the switch again to turn it OFF

## Running Locally
1. Clone this repository
2. Open `index.html` in your browser
3. That's it!
